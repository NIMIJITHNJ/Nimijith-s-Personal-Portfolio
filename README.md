# Nimijith Nimmi Jayan | Personal Portfolio

**Live site:** [nimijithnj.com](https://www.nimijithnj.com)

Personal portfolio website for Nimijith Nimmi Jayan, Graduate Software Engineer and MSc Artificial Intelligence candidate at Heriot-Watt University, Edinburgh. Built with vanilla HTML, CSS, and JavaScript with no frameworks or dependencies beyond Google Fonts and Font Awesome.

---

## Sections

- **Hero:** Introduction, tech badges, CV download, and availability status
- **About:** Background, education, right to work details, and personal interests
- **Skills:** Technical skills across six categories including programming languages, software engineering, AI and ML, conversational AI and NLP, data and databases, and tools and practices
- **Projects:** Six featured projects including the Multiparty AI-Human Spoken Dialogue System, Hospital Management System, Mini Browser Application, Explainable AI Financial Risk Hedging System (ongoing dissertation), Music Genre Classification, and AI Algorithms and Intelligent Agents
- **Experience:** Professional timeline from India Post (2011 to 2024) through UPS and McDonald's Edinburgh, plus volunteer work at the European Chatbot and Conversational AI Summit 2026
- **Certifications:** Python certification from Illinois Institute of Technology (Grade A+), LinkedIn Learning, and Coursera credentials
- **Contact:** Contact form connected to Google Forms, direct email, LinkedIn, and GitHub links

---

## Tech Stack

- HTML5 (semantic structure)
- CSS3 (custom properties, Flexbox, Grid, animations, scroll reveal)
- JavaScript (mobile nav, IntersectionObserver scroll reveal, form handling)
- Google Fonts: DM Serif Display, DM Sans, JetBrains Mono
- Font Awesome 6.4.0 (icons)
- Contact form submissions via Google Forms hidden iframe

## Deployment

The site is deployed via [Vercel](https://vercel.com) with automatic deployments triggered on every push to the `main` branch. No build step is required as Vercel serves the static files directly.

To update the site, push changes to `main` and Vercel redeploys within approximately 60 seconds.

---

## Local Development

No build tools or package managers are required. Simply open `index.html` in any modern browser:

```bash
git clone https://github.com/NIMIJITHNJ/Nimijith-s-Personal-Portfolio.git
cd Nimijith-s-Personal-Portfolio
open index.html
```

Or use the VS Code Live Server extension for local development with hot reload.
